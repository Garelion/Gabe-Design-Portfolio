Personal & Experimental Work

Island Explorers

Systems Designer

Island Explorers is a 2D top-down light survival game, developed by a team of 3, myself included, for a class assignment during my bachelor's at Full Sail. This is a game where the player must use all available resources to survive and improve their tools to last as long as possible.

Personal Contributions:

  • Design and implementation of the inventory system
  • Design and implementation of the crafting system
  • Design and implementation of player controls and animations

Scraps Level

Level Designer

Scraps was a project made for students to build and add onto, so they could learn to make levels from conception, to blockout, to implementation of mechanics. All assets are original, except the playable character.

Marble Game Level

Designer / Programmer

The Marble Game was a team project from Full Sail that required every team member to design and implement one level per person, as well as working on other mechanics and tasks. I was primarily responsible for the player controls, the collectibles, and a level. All done in a week.

Simple Tanks

Designer / Programmer

Simple Tanks is a game created in a one-day game jam. It's a game where up to 2 players can fight each other in tanks from a top-down 2.5D perspective. In this project, I was responsible for the tank controls and the level designs.

Modeled Scene

3D Artist (Maya)

This was a project where students were tasked with modeling some assets and creating a scene with said assets. All of it was done in Maya during the first few months of my bachelor's degree. I added a game to it so that the viewer can find all of the hidden Spongebobs in the scene.

Gex Lair

3D Artist (Unity)

The Gex Lair Model was a project where students were tasked with recreating a scene from their favorite franchises in Unity. While not 100% the same, the model did come out looking very similar to the original.
